Gaussian processes

A Gaussian process is a set of random variables, any subset of which is multivariate Gaussian, with mean defined by the same mean function and covariance matrix defined by the same covariance function. Let's first plot some Gaussian processes, and then use Gaussian processes for regression.

The simplest Gaussian process: uncorrelated random variables

The simplest mean function is the zero function, , and the simplest covariance function is the identity covariance function, , where is the Kronecker delta function. We'll use these for our first example of a Gaussian process. Suppose we have a set of n points . The identity covariance function does not depend on the points, so the covariance matrix is simply the identity matrix. Now, sampled from is an instantiation of a Gaussian process.
To be concrete, let's choose a 1-dimensional set of regularly-spaced n points , and generate a sample .
n = 100;

The values of are not correlated with each other, and the plot of for the n points is what you would expect.

Gaussian processes with entirely uncorrelated random variables are not at all interesting. But they are useful to serve as this first example.

Correlated random variables

Covariance functions generally assign large covariance to pairs of points that are nearby and small covariance to pairs of points that are far apart. In ML, this means that we assume that a pair of feature vectors and will have more strongly corrrelated target values and , respectively, if the feature vectors are closer together, as measured by the covariance function. Gaussian process regression

We drew the above plots with lines between the points to suggest that an instance of a Gaussian process defines a function. This is exactly our aim, and we can imagine that we can define a function implicitly if we have an infinite set of consisting of all possible points at which we want to define a function.
Before we observe any data that we use for regression, the mean function (which is usually chosen to be the zero function) and the kernel function (along with any parameters such as s) define a set functions. The Gaussian distribution defines the prior probability of each of those functions. You can imagine that a rough function would not be very likely for a Gaussian kernel function with a large length scale.
Once we have observed some data, then the probability of each of the functions changes. This is the posterior probability distribution of the functions. Gaussian process regression chooses the most probable function from this posterior distribution of functions.
We'll illustrate Gaussian process regression for synthetic data, where the true function is .
fun = @(x) sin(2*pi*x);
Generate synthetic data.
sigma2 = 0.1;
x = rand(4,1);
y = fun(x) + sqrt(sigma2)*randn(size(x));
Form covariance matrix.
n = length(x);
K = zeros(n,n);
s = 0.1;
for i = 1:n
for j = 1:n
K(i,j) = exp(-0.5/(s^2) * (x(i)-x(j))^2);
end
end
C = K + sigma2*eye(n);
Calculate predictions.
xx = 0:0.01:1;
yy = [];
vari = [];
for xval = xx
kvec = exp(-0.5/(s^2) * (x-xval).^2);
yval = kvec'*(C\y);
yy = [yy yval];
vari = [vari (1-kvec'*(C\kvec))];
end
Finally, plot the results.
clf
plot(xx,fun(xx),'--'); % true function
hold on
plot(x,y,'ko'); % data points
plot(xx,yy); % mean function
lo = yy-sqrt(vari);
hi = yy+sqrt(vari);
h = patch([xx xx(end:-1:1) xx(1)], [lo hi(end:-1:1) lo(1)], 'b');
set(h, 'EdgeAlpha',0, 'FaceColor',[.8 .7 .6], 'FaceAlpha',.3)
Since we have a predictive distribution, Gaussian process regression can give us estimates of the uncertainty of our predictions.
